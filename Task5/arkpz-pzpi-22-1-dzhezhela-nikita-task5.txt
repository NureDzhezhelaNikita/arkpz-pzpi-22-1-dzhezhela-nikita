3 ХІД РОБОТИ
Для розгортання програмної системи необхідно мати локально встановленні:
* Node 18.21.0+
* Docker
* PNPM Package manager
* Supabase CLI
Після цього необхідно:
1. Створити проєкт у Supabase.
2. Створити та налаштувати таблиці за допомогою sql запиту наведеного у попередніх лабораторних роботах.
3. За допомогою пакетного менеджера pnpm встановити залежності: pnpm install
4. Перейти у директорію cd ./packages/supabase
5. За допомогою Supabase CLI розгорнути Edge Functions
